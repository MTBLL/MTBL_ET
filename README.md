# MTBL Extract-Transform Tool
This tool is the script used to invoke the Extract and Transform processes for MTBL data.

## Tool Invokations
### Extract
- ESPN Extractor
  - /Users/Shared/BaseballHQ/tools/_extract/ESPN_API_Extractor
- Fangraphs Extractor
  - /Users/Shared/BaseballHQ/tools/_extract/Fangraphs_API_Extractor
- Baseball Savant Extractor
  - /Users/Shared/BaseballHQ/tools/_extract/Savant_API_Extractor
  
These all output to the /Users/Shared/BaseballHQ/resources/extract/ directory.

### Transform
- Player Universe Transformer
  - /Users/Shared/BaseballHQ/tools/_transform/Player_Universe_Trx
  
These all output to the /Users/Shared/BaseballHQ/resources/transform/ directory.
